Présentation du service

Marche à suivre en 4 étapes


  1. Importez votre fichier d'adresses, saisissez votre email,
  2. Le service vous envoie un email de prise en charge,
  3. Lorque le traitement est terminé, vous recevez un email vous permettant de récupérer le fichier traité.
  4. Le fichier contiendra vos données ainsi que des colonnes complémentaires (site de distribution, identifiant de tournée,...)

Explications

Fichier

Le fichier peut être de type FSPlus (avec ou sans colonne "Action") ou un format libre.

  • Pour les fichiers "libres" non FSPlus, le fichier csv doit avoir dans l'entête le type des champs : address_row1, ..., address_row6, postal_code, locality,
  • Le séparateur de champs doit être le point-virgule,
  • Le fichier peut contenir des colonnes personnelles.

Informations retournées

Voici un exemple de fichier retourné, les données complémentaires renvoyés par le service sont :
  • deliverable : indique si l'adresse est delivrable (l'adresse existe et une et une seule tournée est rattachée à cette adresse),
  • distribution_site.id : Identifiant REGATE du site de distribution (Ex: 752310),
  • distribution_site.id_roc : Identifiant ROC du site de distribution (Ex: A21588),
  • distribution_site.label : Libellé du site de distribution (Ex: OBERNAI PDC1),
  • round.id : Identifiant unique de tournée (Ex : 123456),
  • round.label : Type de tournée (BP : Boite Postale, CS : Course Speciale, TL : Tournée Lettre, TM : Tournée mixte : la tournée contient une partie Courrier et une partie Colis, TC : Tournée Courrier, KI : Kiosque) avec son libellé (Ex : CS0001),
  • round.rank : Rang de l'adresse dans la tournée (Ex : 3),
  • address_ids.id : Identifiant d'adresse (cea) le plus fin identifié (Ex: 75118224SS),
  • address_ids.type : Type de l'identifiant d'adresse (address_ids.id) le plus fin trouvé : internal_service>cedex_id>row3>row4_number>row4_road>row56 (Ex: row3),
  • address_ids.rowX : Identifiants d'adresse (cea) de niveau rowX,
  • cabin_site.id : Identifiant REGATE du site de cabine (Ex: 752310),
  • cabin_site.id_roc : Identifiant ROC du site de cabine (Ex: A21588).
